Section:
  ops cloud k3s

Usage:
  k3s create <server> [<user>]
  k3s delete <server> [<user>]
  k3s info
  k3s kubeconfig
  k3s status

Commands:
  create     create a k3s with ssh in SERVER=<server> using USERNAME=<user> with sudo
  delete     uninstall with ssh in SERVER=<server> using USERNAME=<username> with sudo
  info       info on the server
  kubeconfig recover the kubeconfig from a K3S server SERVER=<server> with user USERNAME=<username>
  status     status of the server
